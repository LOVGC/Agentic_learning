First Principles · 第一性原理

What is Learning?

把教育学 / 心理学 / 神经科学的定义放下，只留下最不可再分的东西

Learning 是一个有限资源的智能体，通过经验持续更新其内部世界模型， 从环境数据中提取可泛化的规律，以提升对未来的预测能力，从而做出更优决策、提高适应能力的过程。

図

一张图，看懂 Learning 的本质Agent ⇄ Env · 一个学习器 + 一个心智模型 (world model)

Agent ⇄ Env 交互：action / state · reward 两个模型：① 学习器 + ② 心智模型 ① 构建 ② · ② ≈ 环境（预测 s′, r）
1

像 RL：Agent ⇄ Env

学习发生在智能体与环境的交互里：agent 发出 action，env 返回 (state, reward)。 这串经验，就是喂给学习器的原料——没有交互，就没有可学的数据。

2

Agent 内部有两个模型

① 压缩 / 抽象模型是「学习器」：吃进环境的资料，压缩出一个 mental model； ② mental model 是「被学到的东西」。一个负责学，一个负责用。

3

心智模型 ≈ 环境的副本

② 输入 (state, action)、输出对环境的预测 (next state, reward)——它就是 env 的内部仿真。 学习，就是让它的预测越来越贴近真实环境。

1

同一个 loop，发生在所有尺度上

这张 agent–env 图并不只描述某一个层面——换一个 agent、换一段时间，它在所有尺度上反复上演：

个人 · Personal
agent 是一个人
群体 · Collective
agent 是一群人，比如一个国家
短时间 · Short
一个 agent 的一天、一生
长时间 · Long
整个人类的进化史
2

我说的「mental model」，其实是学习的能力

我以前一直强调的 mental model，对应的其实是图里的 1 压缩 / 抽象模型—— 它代表一个 agent 学习新东西的能力。

mental model　=　compressed knowledge　=　abstractions　=　basic framework

所以学一个领域，关键是先找到它的 Basics——一个基本框架 (framework)，让这个领域里其它几乎所有东西， 都只是它的一个实现或变体（an implementation or adaptation of this framework）。

3

已经建好的 mental model = 经验 = 舒适区

而一个 agent 此刻已经拥有的 2 心智模型，代表它已有的经验—— 也就是这个 agent 的舒适区。

于是：学习 = 用 ①（学习的能力）不断把 ②（舒适区）往外推。

4

Learning 的本质，有很多层

  • 在最底层：learning 是生命适应环境的一种机制。
  • 在另一层：learning 本身就很有趣、好玩。

Key insight · 关键

但无论如何，learning 就是这张图所描述的一切。不同的人会用不同的词来描述这张图—— 可本质上，似乎都是同一个意思。

当你忘了怎么学习的时候，就回来看看这张图。
